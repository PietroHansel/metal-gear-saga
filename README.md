# metal-gear-saga

Projeto de desenvolvimento web focado na evolução para Full Stack, apresentando uma linha do tempo completa da **saga Metal Gear**, criada por **Hideo Kojima**. A aplicação organiza eventos e personagens em uma interface estruturada, servindo como <b>laboratório prático</b> para consolidação de habilidades em desenvolvimento e arquitetura web.
